APNを設定します。単一のAPNのみ設定することができます。

すでにインターネットに接続できている状態でAPNを変更した場合、変更を反映させるためにはモジュールを再起動する必要があります。```sudo reboot```などの方法で再起動を行ってください。

```bash
$ sudo candy apn set -n APN名 -u ユーザーID -p パスワード
```
