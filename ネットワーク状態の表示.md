モバイルネットワークの状態を表示します。rssiの単位はdBmとなります。結果文字列のrssiDescには以下の値が入ります。

1. `"OR_LESS"` ... rssiの値以下 であることを示す
1. `"OR_MORE"` ... rssiの値以上 であることを示す
1. `"NO_SIGANL"` ... 圏外
1. `""` ... rssiの数値通り

networkのプロパティには、`ONLINE `、`OFFLINE `または`UNKNOWN `の文字列が入ります。

```bash
$ sudo candy network show
{
  "rssi": "-85",
  "network": "ONLINE",
  "rssiDesc": ""
}
```
