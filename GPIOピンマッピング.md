[LTEPi for D](https://www.candy-line.io/製品一覧/ltepi-for-d/)は以下のGPIOピンを使用しています。

GPIOをお使いの場合は、下記以外のピンをご利用ください。

| RPi B+/2 GPIO  | GPIO HW Pin#  | AMP520/AMM570 | LTEPi for D |
| -------------- | ------------- | ------------- | ------------|
|    20 (OUT)    |     PIN 38    |     POWER     |      -      |
|    21 (OUT)    |     PIN 40    |     RESET     |      -      |
|     6 (OUT)    |     PIN 31    |    RESERVED   |      -      |
|    14 (IN)     |       -       |      TX       |      -      |
|    15 (OUT)    |       -       |      RX       |      -      |
|    16 (IN)     |     PIN 36    |      RI       |      -      |
|     4 (OUT)    |     PIN 7     |       -       |     LED2    |
